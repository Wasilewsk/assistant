# Project-Cortana — For the Blind

A voice-activated AI assistant designed **for visually impaired and blind users**. Inspired by Cortana from Halo, reimagined as an accessibility tool.

## Vision

Voice is the primary interface — no screens, no visual dependencies. Everything is spoken, every action confirmed by sound.

## Features

- **Wake-word activation** — Say "Hey Cortana" to begin
- **Read anything aloud** — Text files, PDFs, Word documents, web pages, clipboard content
- **Web browsing by voice** — Search the web, read summaries, navigate results
- **OCR** — Read text from images and scanned documents
- **System control** — Open apps, control volume, navigate files, shutdown
- **Information services** — News headlines, weather, time/date, Wikipedia, dictionary
- **Productivity** — Timers, alarms, reminders, notes, calculator
- **File navigation** — List directories, read file info, open files
- **Audio cues** — Distinct sounds for different events and responses
- **Speed control** — Adjust reading speed and voice on the fly

## Quick Start

```bash
pip install -r requirements.txt
python app/main.py
```

Say **"Hey Cortana"** to wake the assistant, then speak a command.

## Commands

| Category | Examples |
|----------|----------|
| Wake | "Hey Cortana" |
| Read | "read this file", "read clipboard", "read screen" |
| Web | "search for...", "read Wikipedia about...", "open YouTube" |
| System | "open Chrome", "volume up", "list files", "shutdown" |
| Info | "what's the news", "weather in London", "what time is it" |
| Utility | "set timer 5 minutes", "take a note", "calculate 25 * 4" |
| Help | "help", "what can you do", "list commands" |
| Sleep | "go to sleep", "goodbye" |
